# Automated Resume Parser

A Python-based resume parsing project that extracts useful information from PDF resumes.

## Features
- Extracts text from PDF resumes
- Detects email addresses
- Detects phone numbers
- Identifies common technical skills

## Technologies
- Python
- PyPDF2
- Regular Expressions (re)

## Run

```bash
pip install PyPDF2
python main.py
```

Enter the path of a PDF resume when prompted.
